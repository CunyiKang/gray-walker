# 灰域行者 | Gray Walker

> 一个模拟精神分裂谱系体验的HTML治疗游戏

## 简介

「灰域行者」是一款基于Web的互动治疗游戏，旨在帮助玩家理解和共情精神分裂谱系障碍患者的日常体验。通过模拟幻听、关系妄想、命令性幻听、被害妄想、思维插入等症状，让玩家在安全的虚拟环境中感受这些症状带来的困扰与压力。

**⚠️ 重要声明**：本游戏仅用于教育目的和心理健康宣传，不构成医学诊断或治疗建议。如有心理困扰，请寻求专业帮助。

## 功能特点

- **6个真实场景**：涵盖精神分裂症谱系的多种典型症状
- **压抑程度曲线**：精心设计的情绪节奏，从轻度到最困难再到和解
- **伦理合规设计**：不假设家人伤害，强调大脑误读机制
- **Grounding练习**：每个场景后提供着陆技术练习
- **治疗师记录系统**：完整的数据记录，支持打印报告
- **主题切换**：支持深色/浅色模式
- **纯前端实现**：无需后端，单HTML文件即可运行

## 场景介绍

| 场景 | 时间 | 症状主题 | 压抑程度 |
|------|------|----------|----------|
| 清晨醒来 | 07:15 | 幻听入门 | 轻度 |
| 药效消退 | 09:00 | 命令性幻听 | 中度 |
| 午后课堂 | 14:30 | 关系妄想 | 最困难 |
| 超市购物 | 17:00 | 被害妄想 | 缓解 |
| 家庭晚餐 | 19:00 | 思维插入 | 轻度 |
| 深夜和解 | 23:00 | 内心对话 | 和解 |

### 压抑程度曲线

```
轻度 ──→ 中度 ──→ 最困难 ──→ 缓解 ──→ 轻度 ──→ 和解
```

设计理念：让玩家逐步体验症状，在最困难后给予缓解，最终以内心和解结束，避免留下过度负面情绪。

## 游戏玩法

1. **开始游戏**：点击"开始旅程"进入第一个场景
2. **体验场景**：阅读文字描述，点击"继续"推进剧情
3. **应对幻觉**：在幻觉出现时选择如何应对
4. **Grounding练习**：完成场景后的着陆技术练习
5. **查看报告**：游戏结束后查看完整的治疗师记录

### 操作说明

- 点击对话框继续剧情
- 幻觉声音会在右侧面板显示
- 底部显示当前情绪状态
- 可随时点击"查看报告"查看进度

## 设计理念

### 伦理考量

本游戏在设计时特别注意伦理问题：

1. **不妖魔化家人**：场景2中，原设计假设"妈妈说负面话语"，已修正为"妈妈轻声说了句什么，但你没听清，大脑自动填充了让你不安的内容"。这强调是**大脑误读**而非家人伤害。

2. **症状真实性**：所有症状描述基于真实临床案例，经精神科医生审核。

3. **安全边界**：游戏提供充足的退出点和心理支持提示。

### 治疗价值

- **共情训练**：帮助家属、医护人员理解患者体验
- **去污名化**：让公众了解精神疾病并非"疯癫"
- **自我认知**：患者可通过游戏认识自己的症状模式
